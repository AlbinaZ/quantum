Запуск сервиса

1. Открыть файл settings.py и прописать хост и порт к которому надо прибиндиться

2. Открыть командную строку в этой папке и набрать
	"quipper\Scripts\python" server.py

Работа с сервисом

1. Исходный код -> схема:
	GET запрос с параметрами
	url = "/str_to_circuit"
	params = {"source_codes": JSON сериализованный список строк}
	
	ответ придет как JSON-сериализованная структура ранее обговоренного формата
	Пример ответа сервиса:
	
	'[{"labels": ["b", "a", "c"], "name": "f", "circuit": [["H", 0], ["CNOT", 1, 2], ["H", 1]]}, {"labels": ["b", "a", "c"], "name": "z", "circuit": [["H", 0], ["CNOT", 1, 2], ["H", 1]]}]'

2. Исходный код -> файл:
	GET запрос с параметрами
	url = "/str_to_file"
	params = {
		"source_code": текст программы,
		"extension": расширение файла, поддерживаются форматы eps, pdf, png, ps, svg, svgz.}

	в ответ придет поток байтов.

3. Схема -> текст
	GET запрос с параметрами
	url = "/circuit_to_str"
	params = {"circuit": JSON-сериализованная структура схемы (Не список)}

	В ответ придет строка

4. Схема -> файл
	GET запрос с параметрами
	url = "/circuit_to_file"
	params = {
		"circuit": JSON-сериализованная структура схемы (Не список),
		"extension": расширение файла, поддерживаются форматы eps, pdf, png, ps, svg, svgz.}

	в ответ придет поток байтов.
